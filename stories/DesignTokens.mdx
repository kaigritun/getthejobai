# Design Tokens

GetTheJobAI uses a minimal, dark-first design system with **emerald** as the accent color.

## Colors

### Accent (Emerald)
| Token | Value | Usage |
|-------|-------|-------|
| Primary | `emerald-600` | Buttons, CTAs |
| Hover | `emerald-500` | Button hover state |
| Focus Ring | `emerald-500` | Input focus |
| Success BG | `emerald-900/20` | Success states |
| Success Border | `emerald-800` | Success borders |
| Success Text | `emerald-400` | Success messages |

### Backgrounds
| Token | Value | Usage |
|-------|-------|-------|
| Page | `#0c0c0c` | Main background |
| Card | `white/[0.02]` | Card backgrounds |
| Card Hover | `white/5` | Card hover state |
| Input | `white/5` | Input backgrounds |

### Borders
| Token | Value | Usage |
|-------|-------|-------|
| Default | `white/10` | Cards, inputs, dividers |
| Hover | `white/20` | Card hover states |
| Accent Hover | `emerald-500/50` | Interactive element hover |

### Text
| Token | Value | Usage |
|-------|-------|-------|
| Primary | `white` | Headings, important text |
| Secondary | `white/60` | Body text |
| Muted | `white/50` | Descriptions |
| Subtle | `white/40` | Footer text, timestamps |

## Typography

### Font Stack
- Primary: System fonts (`-apple-system, BlinkMacSystemFont, 'Segoe UI', ...`)

### Scale
| Name | Size | Weight | Usage |
|------|------|--------|-------|
| h1 | 2.5rem | bold | Page titles |
| h2 | 1.875rem | bold | Section headings |
| h3 | 1.25rem | semibold | Card titles |
| body | 1rem | normal | Body text |
| small | 0.875rem | normal | Secondary text |

## Spacing

Base unit: 4px

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Tight spacing |
| sm | 8px | Compact elements |
| md | 16px | Default spacing |
| lg | 24px | Section padding |
| xl | 32px | Large sections |
| 2xl | 48px | Major sections |

## Border Radius

| Token | Value | Usage |
|-------|-------|-------|
| sm | 6px | Buttons, inputs |
| md | 8px | Cards |
| lg | 12px | Large cards |
| xl | 16px | Hero sections |

## Transitions

| Token | Value | Usage |
|-------|-------|-------|
| fast | 150ms | Hover states |
| default | 200ms | Most transitions |
| slow | 300ms | Layout changes |

## Micro-interactions

- **Hover scale**: `scale-[1.02]`
- **Active scale**: `scale-[0.98]`
- **Loading spinner**: 4px border, `animate-spin`
